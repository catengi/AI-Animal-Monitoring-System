**ระบบเฝ้าระวังสัตว์ป่าด้วยปัญญาประดิษฐ์ (AI Animal Monitoring System)
**
**การติดตั้งใช้งาน**
1. Download File ทั้งหมด
2. Run ไฟล์ชื่อว่า app.py เพื่อเริ่มต้นการทำงาน (รันผ่าน Python Runner / Python IDE ได้)
3. สามารถทดสอบการใช้งานได้โดยใช้วิดีโอทดสอบที่แนบไปพร้อมไฟล์โปรแกรม
4. หากต้องการนำผลลัพธ์ไปเชื่อมต่อกับระบบภายนอกให้ใช้ REST API Get Request ไปที่ http://localhost:5000/status เพื่อรับผลลัพธ์จากการตรวจจับได้เลย
